# Zelda-Like

Input : 
- E : interaction
- F : attaque de faux
- A : lancer de faux
- R : lancer de poids

Directions : 
- ZQSD
- Flèches


Manette : 
- A : interaction
- X : attaque de faux
- B : lancer de faux
- Y : lancer de poids







Si vous êtes bloqué, lancez votre poids à droite du tronc du cerisier.
